# Пакет geometric_lib

### Пакет, используемый для вычисления площадей и периметров простейших геометрических фигур: круг, прямоугольник, квадрат, треугольник

Не требует установки сторонних пакетов. Содержание пакета:


---
# Модуль circle
* ### Функция area(r) : возвращает (число) площадь круга радиуса r 

**Параметры:**

* **r (число)** : радиус круга

**Пример использования:**
```
>>> geometric_lib.circle.area(10)
314.1592653589793
```
---
* ### Функция perimeter(r) : возвращает (число) длину окружности радиуса r

**Параметры:**

* **r (число)** : радиус окружности

**Пример использования:**

```
>>> geometric_lib.circle.perimeter(10)
62.83185307179586
```
---
# Модуль rectangle
* ### Функция area(a, b) : возвращает (число) площадь прямоугольника с сторонами a и b

**Параметры:**

* **a (число)** : первая сторона прямоугольника
* **b (число)** : вторая сторона прямоугольника (перпендикулярна первой стороне)

**Пример использования:**

```
>>> geometric_lib.rectangle.area(4, 5)
20
```
---
* ### Функция perimeter(a, b) : возвращает (число) периметр прямоугольника с сторонами a и b

**Параметры:**

* **a (число)** : первая сторона прямоугольника
* **b (число)** : вторая сторона прямоугольника (перпендикулярна первой стороне)

**Пример использования:**

```
>>> geometric_lib.rectangle.perimeter(4, 5)
18
```
---
# Модуль square
* ### Функция area(a) : возвращает (число) площадь квадрата со стороной a

**Параметры:**

* **a (число)** : сторона квадрата

**Пример использования:**

```
>>> geometric_lib.square.area(4)
16
```
---
* ### Функция perimeter(a) : возвращает (число) периметр квадрата со стороной a

**Параметры:**

* **a (число)** : сторона квадрата

**Пример использования:**

```
>>> geometric_lib.square.perimeter(10)
40
```
---
# Модуль triangle
* ### Функция area(a, h) : возвращает (число) площадь треугольника с основанием a и высотой h

**Параметры:**

* **a (число)** : основание треугольника
* **h (число)** : высота треугольника

**Пример использования:**

```
>>> geometric_lib.triangle.area(10, 10)
50.0
```
---

* ### Функция perimeter(a, b, c) : возвращает (число) периметр треугольника с сторонами a, b, c

**Параметры:**

* **a (число)** : первая сторона треугольника
* **b (число)** : вторая сторона треугольника
* **c (число)** : третья сторона треугольника

**Пример использования:**

```
>>> geometric_lib.triangle.perimeter(7, 7, 7)
21
```
