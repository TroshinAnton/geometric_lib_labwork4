# Цели и задачи тестирования

Цель: проверка корректности вычислений показателей геометрических фигур на корректных входных данных.

Задачи:
+ Проверка корректности математических вычислений.
+ Подтверждение ожидаемого поведения при вводе больших, а также дробных чисел.
Подтверждение ожидаемого поведения при нулевых входных значениях

# Описание тестируемого продукта
Протестирован будет пакет _geometric_lib_, используемый для вычисления площадей и периметров простейших геометрических фигур: круг, прямоугольник, квадрат, треугольник.*

Его функционал:
```
Функция circle.area(r) : возвращает (число) площадь круга радиуса r

Функция circle.perimeter(r) : возвращает (число) длину окружности радиуса r

Функция rectangle.area(a, b) : возвращает (число) площадь прямоугольника с сторонами a и b

Функция rectangle.perimeter(a, b) : возвращает (число) периметр прямоугольника с сторонами a и b

Функция square.area(a) : возвращает (число) площадь квадрата со стороной a

Функция square.perimeter(a) : возвращает (число) периметр квадрата со стороной a

Функция triangle.area(a, h) : возвращает (число) площадь треугольника с основанием a и высотой h

Функция triangle.perimeter(a, b, c) : возвращает (число) периметр треугольника с сторонами a, b, c
```

# Область тестирования

В область тестирования входят все реализованные в пакете 8 функций, перечисленные в пункте 2.

# Стратегия тестирования
Фреймворк: unittest

Тип тестирования: функциональное

Техника: Black-box тестирование

Запуск с помощью командной строки:
```
python3 -m unittest tests/test_<фигура>.py
```
Для каждой фигуры тесты запускаются отдельно.

# Критерии приемки

Продукт считается прошедшим тестирование при выполнении следующих условий:

+ Все тесты выполнены с положительным результатом.
+ Не наблюдается аварийных завершений.
+ Вычисления совпадают с описанными в тестах значениями.

# Ожидаемые результаты 

Поскольку дополнительно в geometric_lib не учитываются особые случаи, например, применения формул для вырожденного треугольника, или входных данных с отрицательными числами, с прохождением тестов не ожидается проблем.
